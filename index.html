Comment ça marche ?

Création de site Internet

Tech Assist Web conçoit et réalise des sites web clés-en-main, avec une méthodologie en neuf étapes :

Le Besoin

Pour créer un site web, il faut d’abord savoir ce qu’on veut. Tout commence donc par un entretien, afin de comprendre vos besoins. On y abordera les prestations demandées, le contexte, la thématique, le public visé, le calendrier de réalisation souhaité, les contraintes, les concurrents, etc.

Les Contenus

Une fois le besoin clairement identifié, Tech Assist Web étudie la documentation qui lui est fournie par le Client. Charte graphique, offre de services, articles de presse, tous les documents sont explorés et analysés par Tech Assist Web, qui en ressort une proposition d’arborescence du contenu avec ses recommandations.

La Maquette

Tech Assist Web soumet au Client quelques modèles de maquettes de site web en responsive design qui correspondent au secteur d’activité et au type de site web commandé. Le Client sélectionne le template qui lui convient, en soulignant ses préférences concernant la disposition des éléments.

Le Déploiement

Les principaux choix Client étant désormais arrêtés, Tech Assist Web procède à l’installation technique du site. Nom de domaine, hébergement web, emails, certificat SSL, base de données, CMS, modules de sécurité et de maintenance… à la fin de la procédure, les emails sont fonctionnels, et le site web affiche le logo du Client et la mention « en construction ».

L'Intégration

L’intégration est l’étape durant laquelle Tech Assist Web va donner au site vierge installé l’apparence décidée  lors du choix de la maquette. C’est pendant l’intégration que seront mise en forme les contenus (textes, images…) dans un ensemble cohérent et agréable à explorer.

L'Exploration

A la fin de l’intégration, un accès restreint est créé au Client par Tech Assist Web, qui lui permet ainsi de consulter le site web. Il peut alors s’approprier son support, et faire part de ses observations à Tech Assist Web, pour d’éventuelles corrections de contenus ou de forme.

L'Optimisation

Pour garantir au site web un bon référencement naturel, il est nécessaire d’accorder une attention particulière à son optimisation pour les moteurs de recherche (SEO). Cela passe d’abord par de l’optimisation technique (structure du code, vitesse de chargement, poids des pages, compression des images, etc.), puis par de l’optimisation éditoriale (titres, mots-clés, balises métas, etc.)

Le Lancement

Une date est choisie pour le lancement officiel du site web. Ce dernier devient publiquement accessible, et est soumis aux moteurs de recherche pour son indexation. Une annonce est sponsorisée sur la page Facebook de Tech Assist Web avec un lien vers le site web. Enfin, un plan du site est soumis à Google pour indexation.

La Maintenance

Afin de garantir la stabilité du site web, et la sécurité de ses utilisateurs, Tech Assist Web assure une prestation de maintenance technique et sécuritaire continue sur le site web. Enfin, Tech Assist Web propose aussi une formation pour la gestion du contenu de votre site Internet.
